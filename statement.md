# Project Statement

## Problem Statement
Users often want a simple way to store words, their meanings, and optional tags without using large or complex applications.  
This project solves that by providing a lightweight command-line tool that allows users to create, search, and view stored words easily.  
All data is saved in a CSV file so it can be accessed again when the program restarts.

## Scope of the Project
- Provide basic vocabulary management using Python  
- Store data persistently using CSV  
- Allow adding, searching, and listing words  
- Support optional tags for categorization  
- Simple text-based interface for beginners  
The project does **not** include advanced features like cloud sync, multiple user accounts, or GUI.

## Target Users
- Students who want to save and organize vocabulary  
- Beginners learning Python file handling  
- Users who prefer a simple, console-based dictionary tool  
- Anyone needing a lightweight word storage system  

## High-Level Features
- Add new words with meanings and tags  
- Search words by name or meaning  
- Display all saved words in a clean format  
- Persistent storage through CSV  
- Easy-to-use menu-based interface
